генератор используется для создания массива javascript, содержащего список ссылок всех выбранных титров, который также используется для сохранения списка для последующего использования, и позволяет управлять этим списком путем удаления или добавления ссылок, а также позволяет проверить, работают ли эти ссылки

charger fichier : sert a charger les donné qui on éte soit sauvegardé dans le navigateur(navigateur) soit sauvegardé dans un fichier(parcourir). quand a l'option 'inserer liste' elle sert a rajouter une liste dans la liste courante.

save: сохранить данные в браузере и создать в окне ввода данные, которые нужно сохранить, скопированные вставляемые в текстовый файл, потому что вы не можете сохранить данные из браузера во внешний файл; нет меры безопасности, лучше сделать резервную копию в текстовом файле.

generer array : генерирует таблицу, содержащую ссылки названий, и которая должна быть помещена в начало файла javascript игрока, скопировав вставку
