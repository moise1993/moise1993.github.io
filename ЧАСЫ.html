.:: Horloge analogique ::.

Voici une applet permetant de présenter l'heure où la date de différentes façon. Les différents affichages se font par le bias de paramètres à envoyer à l'applet.

Comment utiliser cette applet.

1. Télécharger le fichier horlogeAnalogique.
2. Décompresser le fichier téléchargé dans le répertoire ou se trouve la page HTML ou sera afficher l'applet. Vous obtiendrez 3 fichiers.

  • Le fichier horlogeAnal.java: La source de l'applet.
  • Le fichier horlogeAnal.class: Le fichier compilé de l'applet.
  • La page HTML qui vous présente cette applet.

3. Incorporez l'exemple de code dans votre page HTML :

horlogeAnal.class" width="300" height="50">
" value="7">
" value="0">
" value="4">
